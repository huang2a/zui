导航组件样式

水平导航组件用法说明

1、按照示例代码的结构即可启用一个默认的导航条（默认为不固定）

2、通过最近class类zui-nav-right可以启用一个在右边的导航菜单。还可以在class类zui-nav-item后面追加类zui-nav-nowidth进行清楚导航的最小宽度。（多数用于显示图标导航）参考顶部导航

3、顶部导航栏目固定可通过配置 class 类 zui-nav-fixed 实现，当需要手动去设置 body 的 padding-top:50px;

4、通过zui-nav-child的类可以实现下拉菜单样式

1、tab默认是点击切换的。tab的宽度是自动的。

        弹窗组件用法说明

        1

        名称 类型 默认值 说明
        type（基本层类型） Number 0 可传入的值有：0（信息框，默认）1（页面层）2（iframe层）3（加载层）4（tips层）。 若你采用layer.open({type: 1})方式调用，则type为必填项（信息框除外）
        title（标题） String/Array/Boolean 信息 若你传入的是普通的字符串，如title :'我是标题'，那么只会改变标题文本；若你还需要自定义标题区域样式，那么你可以title: ['文本', 'font-size:18px;']，数组第二项可以写任意css样式；如果你不想显示标题栏，你可以title: false
        content（内容） String/DOM/Array null content可传入的值是灵活多变的，不仅可以传入普通的html内容，还可以指定DOM，更可以随着type的不同而不同。
        skin（样式类名） String null skin不仅允许你传入layer内置的样式class名，还可以传入您自定义的class名。这是一个很好的切入点，意味着你可以借助skin轻松完成不同的风格定制。目前修改并内置了zui-popup-skin-default（主体色）
        area（宽高） String/Array auto 在默认状态下，layer是宽高都自适应的，但当你只想定义宽度时，你可以area: '500px'，高度仍然是自适应的。当你宽高都要定义时，你可以area: ['500px', '300px']
        icon（图标） Number -1（信息框）/0（加载层） 信息框默认不显示图标。当你想显示图标时，默认皮肤可以传入0-6如果是加载层，可以传入0-2。如：layer.alert('酷毙了', {icon: 1});
        btn（按钮） String/Array 确认 信息框模式时，btn默认是一个确认按钮，其它层类型则默认不显示，加载层和tips层则无效。当您只想自定义一个按钮时，你可以btn: '我知道了'，当你要定义两个按钮时，你可以btn: ['yes', 'no']。当然，你也可以定义更多按钮，比如：btn: ['按钮1', '按钮2', '按钮3', …]，按钮1和按钮2的回调分别是yes和cancel，而从按钮3开始，则回调为btn3: function(){}，以此类推。如：

        anim（动画） Number 0 动画全部采用CSS3。这意味着除了ie6-9，其它所有浏览器都是支持的。目前anim可支持的动画类型有0-6
